# cnsite-yihao-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态页面的存储与展示空间。所收录的页面内容可能涉及各类主题，以笔记、工具、展示页等形式存在。

## 目录说明

```
cnsite-yihao-notes/
├── index.html          # 仓库主页（可选的导航页）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example1.html   # 示例页面
│   └── example2.html   # 示例页面
├── assets/             # 页面引用的静态资源（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：所有独立 HTML 页面存放于此，每个文件代表一个完整的页面或功能模块。
- **assets/**：存放页面所需的公共资源文件，按类型分类放置。

## 页面归档说明

- 每个 HTML 页面均为独立可运行的文档，不依赖后端服务。
- 页面内容可能随时间更新或移除，恕不另行通知。
- 归档仅用于学习、笔记或临时展示，不构成任何形式的长期承诺或服务。

## 维护说明
